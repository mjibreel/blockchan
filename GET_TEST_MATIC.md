## âš ï¸ Troubleshooting

### "Faucet says I already received tokens"
- Each faucet has a cooldown period (usually 24 hours)
- Try a different faucet
- Make sure you're requesting for **Amoy** network, not Mumbai (deprecated) or mainnet

### "Tokens not showing in MetaMask"
- Make sure you're on **Polygon Amoy** network (not Mumbai, not Ethereum)
- Refresh MetaMask
- Check your address on PolygonScan (amoy.polygonscan.com) to confirm

### "Need more tokens"
- Each faucet gives ~0.1-0.5 MATIC
- For hackathon, 0.5 MATIC should be enough for many transactions
- If you need more, wait 24 hours and request again
